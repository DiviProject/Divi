686  // Denial-of-service detection/prevention
687  // The idea is to detect peers that are behaving
688  // badly and disconnect/ban them, but do it in a
689  // one-coding-mistake-won't-shatter-the-entire-network
690  // way.
691  // IMPORTANT: There should be nothing I can give a
692  // node that it will forward on that will make that
693  // node's peers drop it. If there is, an attacker
694  // can isolate a node and/or try to split the network.
